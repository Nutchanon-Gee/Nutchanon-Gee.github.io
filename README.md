# Nutchanon Chaiyaratana Portfolio Website

This is a Quarto/GitHub Pages portfolio website for AD 688 Web Analytics.

## Pages included

- About You
- Projects
- CV
- Contact

## How to use

1. Upload this folder to a GitHub repository.
2. If Quarto is installed, run:

```bash
quarto render
```

3. Commit and push the files to GitHub.
4. In GitHub Pages settings, choose the `docs` folder as the publishing source.
5. Submit the GitHub Pages website URL, the GitHub repository URL, and the zipped source/static files on Blackboard.

## Notes

- The `docs/` folder already includes a static HTML version, so it can be used directly for GitHub Pages.
- Replace any placeholder links with your final GitHub Pages and repository links before submitting.
